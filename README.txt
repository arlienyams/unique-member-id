A simple plugin to display a unique customer or member ID

== Description ==

This plugin displays a unique customer or member ID on the users dashboard and on any page/ post by use of a shortcode 
[nyams_uniqueID]

== Installation ==

1. Upload `unique-member-id.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
